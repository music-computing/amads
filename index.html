amads documentation#

This package collects together a variety of algorithms for symbolic music analysis.

The package is its very early stages. The API is subject to change, and many algorithms are not yet implemented, tested, or documented!

For the source code, visit the GitHub repository.

General algorithms#

amads.algorithms.boundary(score)

Given a score, returns the following: (1) If score is not monophonic, we raise an exception (2) If score has notes, we return a a list of tuples containing note offset and its corresponding strength, respectively

amads.algorithms.break_it_up

Make sense of the relationship between the notes in and the hierarchy of a metrical cycle.

amads.algorithms.durdist1

Distribution of durations of notes in a Score.

amads.algorithms.durdist2

Distribution of duration pairs in a Score.

amads.algorithms.entropy(d)

Calculate the relative entropy of a distribution.

amads.algorithms.hz2midi(hertz)

Convert a frequency in Hertz to the corresponding MIDI note number.

amads.algorithms.ismonophonic(score)

Returns if a musical score is monophonic

amads.algorithms.ivdirdist1(score[, weighted])

Returns the proportion of upward intervals for each interval size

amads.algorithms.ivdist1(score[, weighted])

Returns the interval distribution of a musical score.

amads.algorithms.ivdist2(score[, weighted])

Returns the 2nd-order interval distribution of a musical score.

amads.algorithms.ivsizedist1(score[, weighted])

Returns the interval size distribution of a musical score.

amads.algorithms.nnotes(score)

Returns the number of notes in a musical score.

amads.algorithms.pcdist1(score[, weighted])

Calculate the pitch-class distribution of a musical score.

amads.algorithms.pcdist2(score[, weighted])

Returns the 2nd order pitch-class distribution of a musical score.

amads.algorithms.pitch_mean(score[, weighted])

Compute the mean pitch or mean pitch weighted by duration (in beats) score is a Score.

amads.algorithms.scale(score[, factor, dim])

amads.algorithms.segment_gestalt(score)

Given a score, returns the following: (1) If score is not monophonic, we raise an exception (2) If score is monophonic, we return a 2-tuple of lists for clang boundary offsets and segment boundary offsets, respectively

amads.algorithms.skyline(score[, threshold])

Finds the skyline of a musical score.
